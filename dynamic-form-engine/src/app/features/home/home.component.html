Why This Project?

Dynamic Form Engine was created to demonstrate how complex forms can be generated, validated and controlled through configuration instead of hardcoded UI implementations.

📄

JSON Driven Forms

Define form fields, validation rules and behaviors using simple JSON configuration.

⚡

Automatic Form Generation

Generate Angular Reactive Forms dynamically without manually creating each field.

🔀

Conditional Logic

Show or hide fields automatically based on user selections and business rules.

✅

Runtime Validation

Apply validation rules directly from configuration without changing the UI code.

♻️

Reusable Architecture

Support multiple form types using the same rendering engine.

🅰️

Modern Angular

Built with Angular Standalone Components, Reactive Forms and TypeScript.
